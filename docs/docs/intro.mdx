# Introduction

The Qdrant Kubernetes Operator automates the deployment and management of Qdrant
clusters on Kubernetes. The Operator has controllers that manages Custom
Resource Definitions (CRDs) to extend the Kubernetes API for Qdrant clusters.

## Features

- **Automated deployment**: The Operator automates the deployment of Qdrant
  clusters on Kubernetes.
- **Configuration management**: The Operator manages the configuration of Qdrant
  clusters.
- **Scaling**: The Operator can scale Qdrant clusters up or down.
- **Security**: The Operator manages the security of Qdrant clusters.
